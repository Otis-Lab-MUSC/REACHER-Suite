# reacher-hardware-models

[![Status](https://img.shields.io/badge/status-Stable%20v1.0.0-brightgreen)]()
[![Tech](https://img.shields.io/badge/tech-STL%20%2F%20Arduino-blue)]()

3D-printable STL files and Arduino sketches for building a complete head-fixed behavioral rig.

---

## Key Features

- **~25 STL files** across 6 categories: levers, surgery tools, syringe pump, ethernet components, head fixation, and enclosures
- **Arduino pump control sketch** for syringe-based liquid reward delivery
- **Printing specs** and material recommendations

## Categories

| Category | Description |
|:---------|:------------|
| Levers | Operant lever assemblies for behavioral responses |
| Surgery Tools | Head-bar implantation and surgical positioning aids |
| Syringe Pump | Housing and mechanical components for liquid reward delivery |
| Ethernet Components | Cable management and pass-through mounts |
| Head Fixation | Mounts and clamps for head-fixed recording |
| Enclosures | Rig panels and enclosure components |

## How It Fits

`reacher-hardware-models` provides the physical foundation of the REACHER rig. The 3D-printed components work together with the Arduino UNO running `reacher-firmware` to create the complete behavioral apparatus for head-fixed operant conditioning experiments.
