# pynapse

[![Status](https://img.shields.io/badge/status-Alpha%20v0.1.0-orange)]()
[![Tech](https://img.shields.io/badge/tech-Python%20%2F%20NumPy-blue)]()

Toolkit for preprocessing and analyzing two-photon calcium imaging data aligned to behavioral events.

---

## Key Features

- **Sample / Population / Project** hierarchy for organizing imaging datasets
- **Preprocessing pipeline**: DFOverF, ZScore, GaussianSmoothing, and custom transforms
- **Peri-event tensor extraction** aligned to behavioral timestamps
- **Event parsing** from REACHER session logs
- **Clustering and visualization** utilities

## Tech Stack

- Python / NumPy / SciPy
- HDF5 data storage
- Matplotlib visualization

## How It Fits

`pynapse` sits at the center of the analysis layer. It takes behavioral event logs exported by `reacher` and ROI masks generated by `roigbiv`, aligns neural traces to behavioral events, and produces peri-event tensors that feed into `axplorer` for visualization and statistical analysis.
