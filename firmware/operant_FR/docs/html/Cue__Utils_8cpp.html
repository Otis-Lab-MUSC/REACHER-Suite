Plays a connection or disconnection jingle based on the connection status.

Plays a connection or disconnection jingle.

This function generates an audible feedback using tones to indicate whether a device has been linked or unlinked from a GUI. The pitch increases for connection and decreases for disconnection.

Parameters
connectedA String indicating the connection status ("LINK" or "UNLINK").
cueReference to a Cue object controlling the speaker output pin.

Manages the state of a cue based on time intervals.

Controls the cue speaker by turning it on or off depending on whether the current time falls within the cue's designated on/off timestamp interval. The cue must be armed for this function to take effect.
