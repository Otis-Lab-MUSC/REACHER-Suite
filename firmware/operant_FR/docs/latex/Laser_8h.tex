\doxysubsection{Detailed Description}
Defines the \doxylink{classLaser}{Laser} class for controlling a laser device. 

The \doxylink{classLaser}{Laser} class extends the \doxylink{classDevice}{Device} class to manage a laser with stimulation periods, frequencies, and cycling behavior.
