\doxysubsection{Detailed Description}
Defines the \doxylink{classCue}{Cue} class for managing a tone-\/based cue speaker. 

The \doxylink{classCue}{Cue} class extends the \doxylink{classDevice}{Device} class to handle a speaker that plays tones at specified frequencies and durations, with timestamp-\/based control.
